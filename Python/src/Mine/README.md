# Lingua Franca Mine Automation
This directory contains experiments in designing mining automation systems with multiple interacting agents.

## Install Lingua Franca
To install Lingua Franca, see https://lf-lang.org.

## Run the Examples
Use the Epoch IDE or VS Code plugin or compile the program on the command line (assuming that `lfc` is on the `PATH`):
```bash
lfc PhosphateMine.lf
```
To run the program, follow the instructions on the terminal.
